Typora基本快捷键

Typora

一、快捷键

1.标题 : ctrl+数字

ctrl+1 : 一级标题

ctrl+2 : 二级标题

使用#可以表示标题，一级标题对应一个#，二级标 题对应两个#号，最多至六级标题。在Typora中，#后要紧接着一个空格才能表示标题，否则就是普通字符。

2.下划线 : ctrl+U

3.加粗 : ctrl+B

加粗内容 : 前后两个”**”

4.斜体 : ctrl+l

斜体前后两个”*”

5.选中某句话 : ctrl+L

6.选中某个单词 : ctrl+D

7.搜索 : ctrl+F

8.搜索并替换 : ctrl+H

9.删除线 : alt+shift+5

前后两个”~~”

10.插入图片 : ctrl+shift+i

11.插入链接 : ctrl+K

12.文字高亮 :

前后两个”==”

13.公式块 : ctrl+shift+M或者开头”$”

14. 代码块 : 三个”~~”+enter

15.跳转到文章开头 : ctrl+home

16.跳转到文章结尾 : ctrl+end

17.角标[+^+5+]

18.撤销 : ctrl+Z

19.文件列表及提纲 : ctrl+shift+L

20.创建表格 : ctrl+T

21.选中相同格式的文字 : ctrl+E

22.引用 : ctrl+shift+Q

23.分割线 : 输入***或者—再按回车即可绘制一条水平线
